# Images du template `hairdresser-landing` (v2 — dark luxe)

Le template est **utilisable sans aucune image binaire** : icônes en SVG inline, emplacements photo en dégradés CSS (le look sombre/doré fonctionne sans photo). Pour un rendu final « boutique », ajoutez vos photos ici et remplacez les emplacements dans `index.html` / `css/style.css`.

## Photos recommandées
- `hero.jpg` — intérieur ou portrait du salon, lumineux sur fond sombre, 1600×900+. Remplace `.hero-showcase` (aujourd'hui des visuels dégradés par onglet).
- `creation-1.jpg` … `creation-4.jpg` — transformations (balayage, chignon, coloration, dégradé), ratio 4/3. Remplacent les `.cr .ph`.
- `team-1.jpg` … `team-3.jpg` — portraits des stylistes (ratio 1/1) ; sinon les avatars initiaux suffisent.
- `favicon.png` / `og.jpg` — partages sociaux (référencés dans le `<head>`).

## Règles
- Nommez les fichiers comme ci-dessus ou mettez à jour les chemins dans le HTML/CSS.
- Poids : visez < 200 Ko par photo (WebP/JPEG compressé) pour préserver la rapidité.
- Sur fond sombre, privilégiez des photos lumineuses / haute définition pour ressortir (léger contours dorés conseillés).
- `alt` descriptif en français ; photos libres de droits.
